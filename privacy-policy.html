美客多助手隐私权政策

生效日期：2026-07-15

1. 产品说明

美客多助手是一款面向 Mercado Libre Global Selling 卖家的 Chrome 扩展，用于帮助用户在已登录的 Mercado Libre Global Selling 店铺后台中加载、筛选、缓存并导出自有店铺的商品、订单和备货数据。

2. 我们处理的数据

扩展可能会在用户主动使用功能时处理以下数据：

  • 店铺账号信息，例如 CNID、站点、店铺账号标签。
  • 商品数据，例如商品 ID、ML 编码、标题、图片链接、条码、国家标签、站点条码。
  • 订单数据，例如包装号、销售日期、国家或地区、订单状态、单位、净收益、SKU。
  • 备货数据，例如备货单号、仓库编码、创建日期、产品 ID、产品名称、补货数量。
  • 用户在扩展内设置的筛选条件、加载进度、导出历史记录。

扩展不会收集用户密码，不会读取非 Mercado Libre Global Selling 网站的数据。

3. 数据用途

上述数据仅用于：

  • 在扩展界面中展示商品、订单和备货数据。
  • 根据用户选择的筛选条件加载对应数据。
  • 生成 CSV 或 XLSX 文件供用户下载。
  • 在本地保存最近的导出历史，方便用户后续重新下载。
  • 恢复上次任务状态或筛选条件。

扩展不会将数据用于广告投放、用户画像、信用评估、转售或与产品单一用途无关的目的。

4. 数据存储

扩展使用 Chrome 本地存储保存筛选条件、任务状态、导出结果和导出历史记录。这些数据保存在用户本机浏览器中。

用户可以在扩展的“数据导出”页面清除导出历史，也可以通过 Chrome 移除扩展以删除相关本地数据。

5. 数据共享

美客多助手不会出售、转让或共享用户数据给广告平台、数据经纪商或其他第三方。

扩展不会把用户的店铺数据上传到开发者服务器。扩展只会在用户已登录 Mercado Libre Global Selling 的情况下，与 https://global-selling.mercadolibre.com/* 进行必要通信，以读取用户自有店铺的业务数据。

6. 权限使用说明

扩展使用以下权限来实现核心功能：

  • cookies：读取 Mercado Libre Global Selling 登录会话和 CSRF/XSRF 相关令牌，用于访问用户已登录店铺的数据接口。
  • storage 和 unlimitedStorage：在本地保存筛选条件、任务状态和导出历史记录。
  • downloads：在用户点击下载时导出 CSV 或 XLSX 文件。
  • tabs：打开或聚焦扩展工作台，并检测是否存在已登录的 Mercado Libre Global Selling 标签页。
  • scripting：在必要时复用 Mercado Libre Global Selling 页面上下文读取数据接口。
  • declarativeNetRequestWithHostAccess：为 Mercado Libre Global Selling 接口请求补充必要的 Origin 和 Referer 请求头。
  • host_permissions：仅访问 https://global-selling.mercadolibre.com/*。

7. Limited Use 声明

美客多助手对用户数据的使用遵守 Chrome Web Store 用户数据政策，包括 Limited Use 要求。扩展只会收集、使用或处理实现其单一用途所必需的数据，即帮助 Mercado Libre Global Selling 卖家加载、筛选、缓存并导出自有店铺数据。

8. 安全说明

扩展通过 HTTPS 与 Mercado Libre Global Selling 通信。扩展不会通过 HTTP 传输用户数据，也不会将用户数据发送到第三方服务器。

9. 政策更新

如果扩展功能或数据处理方式发生变化，本隐私权政策会同步更新。

10. 联系方式

如需咨询隐私权政策或数据处理问题，请通过 Chrome Web Store 开发者资料页提供的联系方式与开发者联系。
