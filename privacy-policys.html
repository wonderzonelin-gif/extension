CFTEG登录助手隐私权政策

生效日期：2026-07-23

1. 产品说明

CFTEG登录助手是一款面向供应链管理系统用户的 Chrome 扩展，用于从供应链管理系统加载店铺登录数据，在用户访问匹配的电商平台登录页面时自动填充账号和密码，提升登录效率。扩展支持供应链管理系统（CFTEG/SCXWCK）、速卖通、1688、淘宝、天猫、亚马逊、eBay、Shopee、拼多多、抖音小店等平台。

2. 我们处理的数据

扩展会从供应链管理系统（CFTEG/SCXWCK）配置好的地址加载以下数据：

  • 店铺登录信息，包括店铺名称、平台类型、登录页面地址、账号及加密后的密码。

扩展不会收集用户的浏览记录、Cookie 内容或其他网站的个人数据。密码在传输和存储过程中均以加密形式存在。

3. 数据用途

上述数据仅用于：

  • 在用户访问匹配的电商平台登录页面时，自动填充账号和密码，完成登录操作。
  • 在扩展界面中展示待登录店铺信息。

扩展不会将数据用于广告投放、用户画像、信用评估、转售或与产品单一用途无关的目的。

4. 数据存储

扩展使用 Chrome 本地存储临时保存从供应链管理系统加载的登录数据，用于在匹配的登录页面自动填充。这些数据仅在浏览器本地临时存储，登录完成后即被清除。

用户可以通过 Chrome 移除扩展以删除相关本地数据。

5. 数据共享

CFTEG登录助手不会出售、转让或共享用户数据给广告平台、数据经纪商或其他第三方。

扩展不会把用户的登录数据上传到开发者服务器。扩展仅从供应链管理系统（CFTEG/SCXWCK）接收登录数据，并在本地匹配的登录页面自动填充，不与第三方共享数据。

6. 权限使用说明

扩展使用以下权限来实现核心功能：

  • activeTab：访问当前标签页，以便在登录页面自动填充账号密码。
  • storage：在本地临时存储从供应链管理系统加载的登录数据。
  • host_permissions：仅与供应链管理系统（CFTEG/SCXWCK）通信接收登录数据，以及在匹配的电商平台登录页面执行自动填充。

7. Limited Use 声明

CFTEG登录助手对用户数据的使用遵守 Chrome Web Store 用户数据政策，包括 Limited Use 要求。扩展只会收集、使用或处理实现其单一用途所必需的数据，即从供应链管理系统接收登录数据并在匹配的电商平台自动填充。

8. 安全说明

扩展通过 HTTPS 与供应链管理系统通信接收登录数据。密码在传输和存储过程中均以加密形式存在。扩展不会通过 HTTP 传输用户数据，也不会将用户数据发送到第三方服务器。

9. 政策更新

如果扩展功能或数据处理方式发生变化，本隐私权政策会同步更新。

10. 联系方式

如需咨询隐私权政策或数据处理问题，请通过 Chrome Web Store 开发者资料页提供的联系方式与开发者联系。
